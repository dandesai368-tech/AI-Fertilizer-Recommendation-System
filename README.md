# AI Fertilizer Recommendation System

## Description
An AI-based web application that recommends the best fertilizer based on soil nutrients and environmental conditions using Machine Learning.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Google Colab
- GitHub

## Features
- Predicts the best fertilizer
- User-friendly web interface
- Machine Learning model
- Real-time prediction
